# Fishing Time

### Requirements
- [CMake](https://cmake.org/download/) 3.0+
- [Qt5](https://www.qt.io/download)

### Build and run

```console
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_PREFIX_PATH="/path/to/qt/"
$ cmake --build .
$ ./bin/fishing-time
```
